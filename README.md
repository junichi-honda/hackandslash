# はじめてのハクスラ

ブラウザで動作するハクスラ放置RPGゲームです。

## 特徴

- ⚔️ オートバトルシステム（個別攻撃判定・ダメージポップアップ）
- 📈 経験値システム（敵を倒してレベルアップ、HP全回復）
- 💎 レアリティ別アイテムドロップ（Common / Rare / Epic）
- 🎒 装備管理システム（武器・盾・鎧・指輪・靴）
- 🐉 ボス戦（ステージ5、10にボス登場）
- 🎨 ステージごとに変わる背景色
- 💡 初心者向けチュートリアルポップアップ
- 🏃 ゲームスピード調整（x1 / x2 / x4）
- 💾 オートセーブ機能
- 🎊 ゲームクリア演出（紙吹雪）

## 遊び方

`index.html` をブラウザで開くだけでプレイできます。

### 基本操作

- **装備**: インベントリのアイテムをクリックして装備
- **破棄**: 装備中アイテムの「破棄」ボタンで破棄
- **一括破棄**: Common / Rare アイテムを一括破棄可能
- **速度変更**: 上部の「速度 x1」ボタンでゲーム速度を変更

### ゲームシステム

- 全10ステージ、各ステージ5ウェーブ
- ステージアップ時にHP全回復
- 死亡時はウェーブ1に戻る（ステージは維持）
- アイテムドロップ率: 40%
- 初回敵撃破時は100%武器ドロップ
